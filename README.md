# 간단한 TodoList 만들기 연습
간단한 일정 등록 및 삭제를 위한 TodoList를 만들어 보았습니다.

## 사용방법
![readme](https://github.com/user-attachments/assets/172384ba-2c9e-46b7-b983-2e32e3b896d0)

### 1. 일정 입력하기
사용자의 일정을 입력합니다.

### 2. 등록 버튼 누르기
등록 버튼을 눌러 일정을 등록합니다.

### 3. 등록된 일정 확인하기
파란색으로 표시한 부분에 등록한 일정이 표시됩니다.

### 4. 삭제 버튼으로 삭제하기
삭제 버튼을 눌러 필요하지 않은 일정이거나, 완료한 일정을 삭제할 수 있습니다.
